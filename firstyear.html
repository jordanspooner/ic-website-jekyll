CO161 : Laboratory I

Haskell

1 : Sequences
Finds the nth term of arithmetic and geometric sequences and series

2 : Cryptography
Implements public-key (RSA) and symmetric-key (ECB, CBC) cryptography

3 : Macroprocessor
A simple text macroprocessor

4 : L-Systems
Generates colourful, optionally probabilistic, fractal images (e.g. dragon curve, Koch snowflake, trees etc.)

5 : Calculus
Evaluates symbolic differential calculus

Java

1 : Number Theory
Implements the Euclidean algorithm, and a primality checker

2 : Sequences, Frequency Tables and Matrices
Generates fibonacci numbers, generates frequency tables and provides utilities for working with matrices

3 : Pawn Race
A simple chess minigame, with a computer player that uses alpha-beta pruning to select optimal moves

4 : Picture Processing
A picture processing library that is able to load, invert, grayscale, rotate, flip, blend, blur, mosaic, and save images

5 : Functional Programming
Uses streams in Java to determine geometry of shapes

6 : Turtle Interpreter
Interprets a simple Turtle Graphics Language and outputs the result to a file

7 : Discrete Event Simulation
Simulates random arrivals and processing time, and calculates mean queue length

8 : Spreadsheet
A spreadsheet program able to perform simple calculations

9 : Concurrent Pacman
A concurrent implementation of Pac-Man

10 : Concurrent Matrix Multiplication
Heavily optimised matrix multiplication

11 : Red Black Tree
Implements a red black tree

Assembly

1 : Matrix Multiplication
Implements matrix multiplication

C

1 : ARM Project
An ARM11 emulator and assembler

2 : Motion-Controlled Game Engine
A fully-documented and tested game engine that uses the OpenCV computer vision library for motion-controlled games using a webcam
